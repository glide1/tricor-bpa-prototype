Prerequisites:

For development:

node version ^0.12
npm version ^2.11

After cloning run

```
>npm install <project directory>
```

To run the server just run `npm run start`

For development you probably want gulp installed.
`>npm install -g gulp`

Then run gulp from the project directory `> gulp`
